标日初级上册语法整理

Posted on 2018-07-03

标日初级上册笔记电子版。我是看新东方唐盾的视频学的，以下是根据先生的课程做的笔记。

第一课（肯/否定句，疑问句）

  1. 名は 名です
    存在句。は是主语助词，
  2. 名は 名ではありません
    否定句。
  3. 名は 名ですか
    疑问句，句尾用句号即可。
  4. 名の名
    可以翻译为“的”，可以用于表示从属关系。

第二课（人/物的代词、连词）

  1. これ/それ/あれは 名です
    これ/それ/あれ为事物代词，分别指（近处的）这个，（不远处的）那个和（远处的）那个。
    • これは　雑誌です。
    • それは　カメラ　ではありません。
    • あれは　パソコンですか。
      どれ表示疑问词，翻译为“哪个”。
  2. この/その/あの 名は 名です
    この/その/あの是连体词，用于连接体言（名词代词等）。意思还是（近处的）这个，（不远处的）那个和（远处的）那个。疑问词为どの。
    • この　ペンは　私の　です。
    • あの　新聞は　先生の　ですか。
  3. 名は 名の名です
    有时连接两个名词，不用翻译。
    • 日本語の本 （日语书）
    • その英語の辞書 （那本英 语词典）
  4. 疑问代词
    • 人称代词
      だれ（普通）：谁
      だなた（礼貌）：哪位
      どちら様（尊敬）：哪位
    • 何（なん）：什么
      区别于どれ/どの：哪一个
    • これは 何ですか。
    • 日本語の辞書は どれですか。

第三课（地点的代词、连词）

  1. ここ/そこ/あそこ は 名です
    这里/那里/（远处的）那里是（某场所）。
  2. 名はどこですか
    （某地点）是哪里？
  3. 名も です。
    も是也的意思，可以代替那里原本的助词，例如は等。
    • 私は中国人です。張さんも中国人です。
  4. 名は 名ですか 名ですか。
    选择疑问句。
    • かばん売り場は　一階ですか、二階ですか。
    • 今日は　水曜日ですか、木曜日ですか。

第四课（存在句）

  1. 地点に 人/物が　あります/います
    存在句，某地有某人/物。人/动物用います，物品用あります。

    • 部屋に机があります。
    • あそこにだれがいますか。

  2. 人/物は 地点に　あります/います
    存在句，和上一句相同，只不过地点和存在的人/物顺序是反的。

    • いすは　部屋に　ありません。
    • 子供は　公園に　います。
    • 駅は　どこに　ありますか。
      （口语）駅は　どこですか。
  3. 名　と 名
    表示并列，翻译为“和”。　
    • 時計と眼鏡
    • 居間にテレビとビデオがあります。
  4. 疑问词も
    表示否定。
    • 教室に だれも　いません。
    • 何も 食べません。

第五课（时间）

  1. 今 ～時 ～分 です。
    表示时间。现在是~点~分。

    • 今 何時 ですか。
    • 8時 30分 です

  2. 动词ます形
    现在时和将来时：ます
    否定形：ません
    过去式：ました
    过去否定：ませんでした

  3. 时间に
    表示在某个时间做某事。只有具体的时间点才需要加に，上周、今天、明年等都不需要加に。

    • 学校は　8時半　に　始まります。
      学校8点半开始上课。

  4. から　まで
    从~到~，既可以表示时间，也可以表示空间。
    から和まで都可以单独使用。

    • 私は　9時から　5時まで　働きます。
    • 私は　10時から　勉強します。
    • 森さんは　2時まで　働きます。
  5. いつ　动ますか
    • 試験は　いつ　始まりますか。
    • 仕事は　何時に　終わりますか。

第六课（来、去移动）

  1. 场所 へ 动
    へ是表示移动方向的助词。一般后面跟的动词是移动性动词，最常用的是行きます，来ます，帰ります。

    • 吉田さんは 来週 北京へ帰ります。
    • 李さんは　どこへ　行きましたか。
      へ可以替换成に，へ强调方向，に强调目的。如果是一件事情，那后面只能接に。

  2. 场所 から 动
    从某地来/去/…

    • あの方は　どこから　来ましか。

  3. 名(人) と 动
    表示和某人一起做某事。

    • 小野さんは　友達と　帰りました。
    • 李さんは　だれと　日本へ　来ましたか。

  4. 名(交通工具)で
    で表示使用某种工具。如果是交通工具+で，表示乘坐该交通工具。

    • 上海まで　飛行機で　行きます。

  5. から　まで

    • 李さんは駅からアパートまで歩いて帰りました。

第七课（动词的基本用法）

  1. 名 を 动
    主-宾-谓结构。
    • 李さんは 毎日 コーヒーを 飲みます。
  2. 名(场所)で 动
    • 図書館で　日本語を勉強します。
      今日　どこで　新聞を　読みましたか。
  3. 名 か 名
    表示不确定，可以翻译为“或”
    • -休みは　何曜日ですか。
      -月曜日か火曜日です。
  4. 名をください
    请给我~。
    • コーラとケーキをください。

第八课（授受关系）

  1. 名(工具) で 动
    工具大致可分为三种。
    自身型：用手洗，用脚踢
    外部工具：用相机拍照
    技能型工具：用日语写信，用英语唱歌

    • 李さんは　日本語で　手紙を　書きます。
    • 手紙を　速達で　送ります。
    • 新聞紙で　紙飛行機を　作りました。

  2. 人物1は 人物2に 名(物品)をあげます
    授受关系。人物1给了人物2某物。

    • 弟は　小野さんに　花を　あげます。
    • 母は　長島さんに　ワインを　あげます。
    • 私は　先生に　日本語を　習います。
      授受关系都可以用该句型表达，给的可以不是具体的物体，あげます也可以替换为别的表示授受关系的动词。

  3. 人物1は 人物2に 名(物品)をもらいます
    人物2后面的に可以替换为から，表示从某人那里(得到)。

    • 私は 小野さんに 辞書を もらいました。
    • 私は 長島さんから 写真を もらいました。

  4. 名(人) に　会います
    与某人见面，助词使用に。

    • 私は　明日　駅で　森さんに　会います。
  5. もう
    已经~。搭配过去式。
    • もう　食べました。
      已经吃过了。
    • もう一度
      表示再一次，而不是已经一次。

第九课（一类形容词）

  1. 名は一类形です。
    一类形容词：以い结尾（但不是所有以い结尾的都是一类形容词）。
    肯定形式：～い
    否定：～くない
    过去式：～かった
    过去否定：～くなかった
    例：高い　高くない　高かった　高くなかった
    肯定形式
    • 四川料理は 辛いです。
    • あの スープは 冷たいです。
      否定形式
    • この　スープは　熱くないです。
      过去式
    • 旅行は　楽しかったです。
    • 昨日は　寒くありませんでした。(礼貌形式)
  2. 一类形+名
    ~い直接修饰名词。
    • 広い国
    • 白い紙
  3. を→は
    は：对比/话题，强调名词
    を：强调动作对象
    • この本は　李さんに　もらいました。
    • 私は　コーヒーは　飲みません。
      咖啡的话我是不喝的。
  4. あまり 一类形/动
    • このスープは　あまり　熱くないです。
    • 試験は　あまり　難しくありませんでした。
  5. 程度副词
    • とても/大変
      都是“很”的意思。とても不如大変程度更高
    • 少し/ちょっと
      都是“一点点”，但是ちょっと比少し量更少。
    • あまり/全然
      都是表示否定，后面接动词或形容词都需要用否定形。
      あまり表示“不怎么”，全然表示“完全不”。

第十课（二类形容词）

二类形容词又称形容动词，变形方式与名词一样。
肯定形式：～です
否定式：～ではありません
过去式：～でした
过去否定：～ではありませんでした
例：静かです，静かではあるませ
静かでした、静かではありませんでした

  1. 名は二类形容词です/でした
    • 京都の紅葉は 有名です。
    • この 通りは 静かでした。　　
  2. 名は二类形容词ではありません/~でした
    • 昨日は暇じゃありませんでした。
    • 町は　にぎやかではありません。
  3. 二类形な名
    二类形容词要加な后才能修饰名词。
    • 奈良は静かな町です。
  4. どんな/どう
    都是表示”怎样”。但是どんな是连词，どう是代词。
    どんな后需要加名词，どう不需要加名词。
    • どんな町（怎样的城市）
    • どうですか（怎么样？）

第十一课（形容词的动词用法）

  1. 人は事/物が一二类形容词です
    人は事/物が できます。
    形容词的动词用法。能力、情感的对象前要用が。
    • 小野さんに歌が好きです。
    • 林さんは犬が怖いです。
  2. 名や名
    • 森さんは　サッカーや　野球が　好きです。
    • 吉田さんは時々　中国や　韓国へ　行きます。
    • お店で野菜や　果物などを買いました。
      など相当于中文中的“等等”，表示列举不完，可以省略。
  3. から和だから
    表示原因、理由。から一般接在一个词后面，だから一般接在句子后面。名词形容词后加です再加から，动词后面加ます再加から。
    • 森さんは　お酒が　好きですから、毎日飲みます。
    • 寒いですから、窓を　閉めます。
    • 明日は休みです。だから　子供と動物園へ　行きます。

第十二课 (比较句)

  1. 名は 名より 形容词です
    肯定的比较句，より：“比”。~比~要更~。

    • 李さんは　森さんより　若いです。

  2. 名より 名のほうが 形容词です
    与~相比，~更加~。

    • 北京より　東京のほうが　暖かいです。

  3. 名は 名ほど 形容词否定形
    ~不如~更~。

    • 東京の冬は　北京ほど　寒くないです。
    • 神戸は　大阪ほど　にぎやかじゃない。

  4. 名の中で 名が 一番形容词です。
    在~(大范围)中，~是最~的。

    • スポーツの中で　サッカーが　一番　面白いです。

  5. 名と 名と どちらが　形容词ですか？
    选择疑问句：~和~哪个更~？

    • 日本語と　英語と　難しいですか。
    • コーヒーと　紅茶と　どっちが好きですか。

第十三课 (量词用法)

总体原则：量词放在名词后，动词前。

  1. 名(数量) 动词

    • 卵を　一個　食べます
    • 教室に学生が四人います。
      也可以说成量词の名词，但是这种用法用得少。

  2. 名(时间) 动词
    表示动作持续的时间。

    • 李さんは毎日　七時間　働きます。
    • 昼　一時間　休みます。

  3. 名(时间)に 名(次数) 动词
    表示频率。固定时间段内出现的次数。

    • 李さんは　一週間に二回プールへ行きます
    • この花は　二年に一度咲きます

  4. 名(场所)へ 动に　行きます/来ます。
    双动词连用时，第一个动词要变为连用体。
    去某地做某事。
    动1 に 动2:动作1是动作2的目的。

    • 午後　郵便局へ　荷物を　出しに　行きます。
    • プールへ　泳ぎに　行きます。
    • 私の家へ　遊びに　来ます。
  5. 量词で
    で相当于“一共”。
    • ケーキは　３個で　五百円です
    • 全部でいくらですか。

第十四课 (动词て形)

鉴于我差不多记熟了动词て形的变法，这里就省略了。

  1. 动词て形 动词
    连接两个动作。

    • 昨日 デパートへ 行って、買い物しました。
    • 家へ 帰って、宿題をします。
    • 図書館へ行って 本を借りて、家へ帰りました。

  2. 动词て形から 动词
    连接两个动作，强调先后关系。
    动作1之后，发生动作2。

    • 毎晩　ラジオを聞いてから、寝ます。
    • 昼ご飯を食べてから　出かけます。
    • 家へ帰ってから、もう一度日本語の勉強をします。

  3. 动て ください
    请~。

    • 住所と名前を 書いてください。
    • この荷物を 航空便で 送ってください。

  4. 名(场所)を 动

    • このバスは 駅前を 通ります。

第十五课 (动词ている)

  1. 动词ている
    表示现在进行时，正在~。

    • 小野さんに　今　新聞を読んでいる。
    • 太郎さんは　コーヒーを　飲んでいる。

  2. 动词ても いいです
    可以做某事。
    类似 ~てもかまいません。做~也是没关系的。

    • 家で仕事をしてもいいです。
    • ここで写真を撮ってもいいです。

  3. 动词ては　いけません
    不可以做某事。

    • 飛行機の中で　タバコを　吸っては　いけません。
      • この部屋に　入っては　いけませんか？
        　- はい、入っていけません。
        回答反义疑问句的时候，“はい”表示否定，“いいえ”表示肯定。

  4. 名(人/物体的附着点)に 动

    • 電車に　乗ります。
      　ボートに乗ります。
    • このいすに　座ってください。
    • 李さんはかばんに書類を入れました。

第十六课 (形容词/名词并列)

  1. 一类形 て 形容词
    一类形容词：~い → ~くて
    • この公園は広くて静かです。
      公园很大而且很安静。
  2. 二类形 で 形容词
    二类形容词+で
    • このパソコンの操作は簡単で便利です。
      这个电脑用起来简单又方便。
  3. 名词 で 名词

第十七课（想要某物/做某事）

  1. 名が欲しいです
    想要某物
    • 何が欲しいですか
      你想要什么呀。
  2. 名を动たいです
    想要做某事。
    动词的想做某事：ます形的词干部分+たい
    类似一类形容词，たい也可以变形：～たくない，～たかった，～たくて
    • 私は映画を見たいです
      我想去看画展。
      此处的を也可以换成が。用を强调谓语动词，用が强调宾语名词。
  3. ~ ませんか
    要不要~？（表示建议）
    • 少し休みませんか
      要不要休息一下。
  4. ~ ましょう
    去~吧。（邀请对方和自己一起做某事）
    • 一緒に行きましょう
      一起去吧。
  5. 疑问词 でも
    表示不论、即使。
    • 何でも（无论什么）
    • だれでも（无论是谁）
    • どこでも（无论哪里）

第十八课（发生变化）

  1. 一类形なります。
    一类形容词去掉い加く。なります表示客观变化
    暑い→暑くなります
    寒い→寒假くなります

    • だいたい暖かくなります。
      天气渐渐地暖起来了。
    • 携带は小さくなりました。
      手机越来越小了。

  2. 一类形します。
    します表示主观变化。
    おおきい→大きくします

    • テレビの音を大きくします。
      把电视的音量调大。

  3. 二类形/名になります。
    同一类，なります表示客观变化。

    • もう元気になりました。
      已经恢复健康了。
    • 息子は医者になります。
      我儿子马上要成为医生了。

  4. 二类形/名にします。
    きれい→きれいにします。

    • 部屋をきれいにしてください。
      请把房间布置得漂亮一点。

  5. 一类形/二类形(+な)ほうがいいです

    • 旅行の荷物は軽いほうがいいです。
      旅行行李还是轻一点好。

第十九课（动词的否定式ない形）

  1. 动词 ない形

    • 一类动词
    • ~+う段 → ~+あ段+ない
      書く → 書かない 読む → 読まない
    • ~+う → ~+わ+ない
      買う → 買わない
    • ある→ない（特殊）
    • 二类动词：去掉る，直接加ない
      食べる → 食べない 見る → 見ない
    • 三类动词
      する → しない　くる → こない

  2. 动词ないでください
    请不要~。

    • 部屋のかぎを忘れないでください。
      不要忘记带房间钥匙。
    • 無理をしないでください。
      不要太勉强。
  3. 动词なければなりません
    动词ない形去掉い，加上ければなりません。算是双重否定。
    必须做某事，不得不做某事。
    • 李さんは今日早く帰らなければなりません。
      小李今天不得不早点回去。
    • すぐに書類を送らないといけません。
      马上把文件发过去。
      此处的ないといけません等价于なければなりません
  4. 动 なくてもいいです
    不~也可以。
    与 动词て形+もいいです（可以~） 类比。
    • 明日は残業しなくてもいいです
      明天不加班也可以。

第二十课（动词的名词化用法）

  1. 名（人） 动词原形 ことができます
    能力/许可。会做某事
    こと指的是前面的这件事，可以把前面的动词短语+ こと看作一个名词，就和名词+ができます一样，表示能做某事。
    除了能力，也可以
    • 日本語ができます（会日语）
      日本語を話すことができます（会说日语）
    • スミスさんはピアノを弹くことができます。
      史密斯会弹钢琴。
    • 写真を撮ることがてきます。
      这里可以拍照。
    • —日本では何歳から結婚することができますか。
      —女の人は16歳から、男の人は18歳からできます
      —在日本多少岁起可以结婚呀？
      —女性16岁，男性18岁可以结婚。
  2. 名は动词原形ことです。
    こと是形式名词，不用翻译。动词原形修饰こと，相当于名词。
    • 趣味は切手を集めることです。
      我的爱好是集邮。
    • 森さんの夢は外国で働くことです。
      森的梦想是在国外工作。
  3. 动词原形前に、~（动作）
    做某事前先把另一件事做了。名词+の前に也表示做某事之前。
    食事の前に 和 食事をする前に 都表示吃饭前。
    • こちらへ来る前に、電話をかけてください。
      来这里前请先给我打电话。
    • 会議の前に、資料をコピーします。
      会议前先把资料复印好。
  4. 疑问词か
    表示不确定
    • 部屋にだれかいますか。（房间里有人吗？）
      部屋にだれがいますか。（房间里有谁？）
  5. 人物 で
    表示完成动作的对象。
    • みんなで餃子を作ります。
      大家一起包饺子。
    • 先月クラス全員でキャンプへ行きました。
      上个月全班同学一起去野营了。

第二十一课（动词过去式た形）

  1. 动词た形
    变形规则与て形相同
    た形是用于表示过去式的简体，ました是礼貌体。
    • 一类动词：
    • ～く → ～いた
      ～ぐ → ～いだ
    • ～す → ～した
    • ～む、ぶ、ぬ → ～んだ
    • ～う、る、つ → ～った
    • 二类动词：去掉る加た
    • 三类动词：する → した、くる → きた
  2. 动词た形 ことが　あります
    表示曾经有做过某事 。用了た形已经表示过去式了，不需要再用ありません。
    ~ありません：没做过~
    ありません简体是ない　たない
    • すき焼きを食べたことがあります。
      我吃过寿喜烧。
  3. 动词た形後で
    类似动词+てから，表示做某事之后。
    • 会社を終わった後で飲みに行きます。
      下班后一起去喝一杯。
    • 映画を見た後で食事をしました。
      看完画展后吃了饭。
    • 仕事の後で、食事をしました。
      工作完以后去吃了饭。
      名词+の後で也是表示之后，与 前に 类似。
  4. 动词た形ほうがいいです。
    最好去做某事。虽然用的是た形，但不是表示做过了这件事，只是建议这样做比较好。
    • もっと野菜を食べたほうがいいです。
      最好多吃点蔬菜。
  5. 动 ましょうか
    提议，与ませんか类似。

第二十二课（礼貌体与敬体）

这一课主要语法点是简体和敬体，礼貌体一般就是ます形，简体是原形，可能还有一些变化。

  1. 动词的简体与敬体






    礼貌体简体
    ます(原形)
    ましたた形
    ませんない形
    ませんでしたなかった
    • ある和行く变形特殊，需要注意。
    • でしたで 的简体是 だった
  2. 一类形容词的简体与敬体
    敬体：一类形容词（包括各种变形）+です
    简体：不加です
  3. 二类形容词的简体与敬体






    简体礼貌体
    ～だ～です
    ～じゃない～ではありません
    ～だった～でした
    ～じゃなかった～ではありませんでした
    1. 名词简体（同二类形容词）
    2. 句子けど句子
      　表示转折或铺垫。
    • -昨日の試験、どうだった？
      -ちょっと難しかったけど、まあまあできたよ。
      昨天的考试怎么样？虽然有点难但还行吧。
    • 来週送别送別会をするけど、都合はどうかな？
      下周要办送别会，大家都有空来吧？
  4. (句子) って
    表示传闻，听说。
    • 太田、今度中国へ転勤だって。
      太田，听说这次你要调到中国了。
  5. ~方（かた）
    人的敬称。あの方：那位。
    动词词干（ます形去掉ます）+方表示方法。
    読みます → 読み方 (读法)
    使います → 使い方 (用法)
    話します → 話し方(说法)

第二十三课（有时A有时B句型）

  1. 动词たり 动词たり します
    表示两个动词并列。有时会做动作一，有时做动作二。用于例举。
    • 小野さんは休みの日、散歩したり　買い物に行ったりします。
      小野休息日有时候会去散步，有时候会去购物。
    • 部屋の中を行ったり来たりします。
      在房间里走来走去。（此处的を表示经过的场所）

  2. 一类形かったり 一类形かったり　です
    二类形だったり 二类形だったり です
    名词だったり 名词だったり　です

    • 飛行機のチケットは　高かったり　安かったり　です。
      机票有的贵有的便宜。

    • この公園は時間によって 静かだったり にぎやかだったり です。
      这个公园根据时间不同，有时安静有时热闹。

    • 日本語の先生は　中国人だったり　日本人だったりです。

  3. 短句 か
    • 短句 + か　どうか
      是否。
    • 私は　今年の夏、北京へ行くかどうか　分かりません。
      不知道今年夏天去不去北京。
    • この料理は辛いかどうか知りません。
      不知道这个菜辣不辣。
    • 小野さんに　仕事が終わったかどうか聞いてください。
      问问小野工作做完了没。
    • 疑问词短句 + か
      表示不确定。
    • かぎが　どこに　あるか　教えてください。
      请告诉我钥匙在哪。
    • 昨日 何を 食べたか 忘れました。
      忘记了昨天吃了什么。
    • どの料理が 辛いか 知りません。
      不知道哪道菜是辣的。

  4. ~に　よって
    根据~
    原形是よる，意思是根据，变成て形连接后一句话。

    • 飛行機の チケットは 季節によって 高かったり 安かったり です。
      机票根据季节时高时低。
    • 結婚式の やり方は国に よって 違います。
      不同国家的婚礼习俗不同。
      +　。—この電車は　いつも　こみますか？
      —いえ、時間に　よります
      电车一直这么挤吗？
      不是的，要看时间。
      此处因为后面没有接句子，所以不用变て形

  5. 动词たり 动词たり します
    一类形かったり 一类形かったり　です
    此类句型结尾处的します和です也可以换成别的说法。

    • 高かったり　安かったり　いろいろ　あります.
      贵的便宜的各种都有。
    • この店の　料理は　味が　濃かったり　薄かったり、毎回　違います.
      这家店的菜口味有时候重有时候轻，每次都不一样。

第二十四课（短句简体形的一些应用）

  1. 短语(简体形)と思います
    这句话一般是用于第一人称做主语，我认为~，我想~。
    如果要用于非第一人称做主语，需要改成“思っています”，就没有主语限制了。
    助词と表示引用。

    • 李さんは　もうすぐ　来る　と思います。
      我想小李马上就要来了。
    • 田中さんは来ないと思います。
      我想田中不会来了。（此处只能用“来ない”，句型要求用简体。）

  2. 名は 短语(简体形)と言いました
    “言いました”表示说了，“言います”表示要说（但还没说）。

    • 陳さんは　パーティーに　行く　と言いました。
      陈先生说他回去派对的。
    • 森さんは　中国語の試験が　難しかった　と言いました
      森说汉语考试很难。
    • 小野さんは 李さんに　ちょっと休みたい　と言いました。
      小野对小李说想休息一下。
      は表示的是主语（说话人），に前的是主语说话的对象（听话人）。
  3. ～のです/～んです
    • ～のです/～んです
      客观事实。～のです多用于书面语，～んです用于口语。
      动词/一类形容词用简体，名词/二类形容词+な
    • すみません　頭が痛いんです。
      抱歉，我头很痛。
    • ー　タバコを やめた ほうがいいですよ。
      ー　そうなんですが、やめることが できないんです
      还是把烟戒了比较好。
      虽然是这样，但是我戒不掉啊。
    • すごい人気ですね。
      ええ、あの歌手は　日本では　とても有名なんです。
      人气好高啊。
      嗯，那是日本很有名的歌手。
    • どうして　～のですか/～んですか
      为什么会这样。
    • ー　どうして　たべないんですか。
      ー　もう　おなかが　いっぱいなんです。
      为什么不吃东西啊？
      我的肚子已经吃得饱饱的了。
    • ー　昨日、どうして　休んだんですか
      ー　頭が痛かったんです。
      昨天怎么请假了？
      头很痛。
  4. ～に　ついて
    有关~。に前一定要放名词。
    • 日本の音楽に　ついて　と思いますか？
      有关日本音乐，你是怎么想的？
