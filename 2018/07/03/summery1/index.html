标日初级上册语法整理

Veröffentlicht am 2018-07-03

（慢慢把笔记整理成电子版,从后往前整理。）

第十六课

  1. 一类形 て 形容词
    一类形容词：~い → ~くて
    • この公園は広くて静かです。
      公园很大而且很安静。
  2. 二类形 で 形容词
    二类形容词+で
    • このパソコンの操作は簡単で便利です。
      这个电脑用起来简单又方便。
  3. 名词 で 名词

第十七课

  1. 名が欲しいです
    想要某物
    • 何が欲しいですか
      你想要什么呀。
  2. 名を动たいです
    想要做某事。
    动词的想做某事：ます形的词干部分+たい
    类似一类形容词，たい也可以变形：～たくない，～たかった，～たくて
    • 私は映画を見たいです
      我想去看画展。
      此处的を也可以换成が。用を强调谓语动词，用が强调宾语名词。
  3. ~ ませんか
    要不要~？（表示建议）
    • 少し休みませんか
      要不要休息一下。
  4. ~ ましょう
    去~吧。（邀请对方和自己一起做某事）
    • 一緒に行きましょう
      一起去吧。
  5. 疑问词 でも
    表示不论、即使。
    • 何でも（无论什么）
    • だれでも（无论是谁）
    • どこでも（无论哪里）

第十八课

  1. 一类形なります。
    一类形容词去掉い加く。なります表示客观变化
    暑い→暑くなります
    寒い→寒假くなります

    • だいたい暖かくなります。
      天气渐渐地暖起来了。
    • 携带は小さくなりました。
      手机越来越小了。

  2. 一类形します。
    します表示主观变化。
    おおきい→大きくします

    • テレビの音を大きくします。
      把电视的音量调大。

  3. 二类形/名になります。
    同一类，なります表示客观变化。

    • もう元気になりました。
      已经恢复健康了。
    • 息子は医者になります。
      我儿子马上要成为医生了。

  4. 二类形/名にします。
    きれい→きれいにします。

    • 部屋をきれいにしてください。
      请把房间布置得漂亮一点。

  5. 一类形/二类形(+な)ほうがいいです

    • 旅行の荷物は軽いほうがいいです。
      旅行行李还是轻一点好。

第十九课

  1. 动词 ない形

    • 一类动词
    • ~+う段 → ~+あ段+ない
      書く → 書かない 読む → 読まない
    • ~+う → ~+わ+ない
      買う → 買わない
    • ある→ない（特殊）
    • 二类动词：去掉る，直接加ない
      食べる → 食べない 見る → 見ない
    • 三类动词
      する → しない　くる → こない

  2. 动词ないでください
    请不要~。

    • 部屋のかぎを忘れないでください。
      不要忘记带房间钥匙。
    • 無理をしないでください。
      不要太勉强。
  3. 动词なければなりません
    动词ない形去掉い，加上ければなりません。算是双重否定。
    必须做某事，不得不做某事。
    • 李さんは今日早く帰らなければなりません。
      小李今天不得不早点回去。
    • すぐに書類を送らないといけません。
      马上把文件发过去。
      此处的ないといけません等价于なければなりません
  4. 动 なくてもいいです
    不~也可以。
    与 动词て形+もいいです（可以~） 类比。
    • 明日は残業しなくてもいいです
      明天不加班也可以。

第二十课

  1. 名（人） 动词原形 ことができます
    能力/许可。会做某事
    こと指的是前面的这件事，可以把前面的动词短语+ こと看作一个名词，就和名词+ができます一样，表示能做某事。
    除了能力，也可以
    • 日本語ができます（会日语）
      日本語を話すことができます（会说日语）
    • スミスさんはピアノを弹くことができます。
      史密斯会弹钢琴。
    • 写真を撮ることがてきます。
      这里可以拍照。
    • —日本では何歳から結婚することができますか。
      —女の人は16歳から、男の人は18歳からできます
      —在日本多少岁起可以结婚呀？
      —女性16岁，男性18岁可以结婚。
  2. 名は动词原形ことです。
    こと是形式名词，不用翻译。动词原形修饰こと，相当于名词。
    • 趣味は切手を集めることです。
      我的爱好是集邮。
    • 森さんの夢は外国で働くことです。
      森的梦想是在国外工作。
  3. 动词原形前に、~（动作）
    做某事前先把另一件事做了。名词+の前に也表示做某事之前。
    食事の前に 和 食事をする前に 都表示吃饭前。
    • こちらへ来る前に、電話をかけてください。
      来这里前请先给我打电话。
    • 会議の前に、資料をコピーします。
      会议前先把资料复印好。
  4. 疑问词か
    表示不确定
    • 部屋にだれかいますか。（房间里有人吗？）
      部屋にだれがいますか。（房间里有谁？）
  5. 人物 で
    表示完成动作的对象。
    • みんなで餃子を作ります。
      大家一起包饺子。
    • 先月クラス全員でキャンプへ行きました。
      上个月全班同学一起去野营了。

第二十一课

  1. 动词た形
    变形规则与て形相同
    た形是用于表示过去式的简体，ました是礼貌体。
    • 一类动词：
    • ～く → ～いた
      ～ぐ → ～いだ
    • ～す → ～した
    • ～む、ぶ、ぬ → ～んだ
    • ～う、る、つ → ～った
    • 二类动词：去掉る加た
    • 三类动词：する → した、くる → きた
  2. 动词た形 ことが　あります
    表示曾经有做过某事 。用了た形已经表示过去式了，不需要再用ありません。
    ~ありません：没做过~
    ありません简体是ない　たない
    • すき焼きを食べたことがあります。
      我吃过寿喜烧。
  3. 动词た形後で
    类似动词+てから，表示做某事之后。
    • 会社を終わった後で飲みに行きます。
      下班后一起去喝一杯。
    • 映画を見た後で食事をしました。
      看完画展后吃了饭。
    • 仕事の後で、食事をしました。
      工作完以后去吃了饭。
      名词+の後で也是表示之后，与 前に 类似。
  4. 动词た形ほうがいいです。
    最好去做某事。虽然用的是た形，但不是表示做过了这件事，只是建议这样做比较好。
    • もっと野菜を食べたほうがいいです。
      最好多吃点蔬菜。
  5. 动 ましょうか
    提议，与ませんか类似。

第二十二课

这一课主要语法点是简体和敬体，礼貌体一般就是ます形，简体是原形，可能还有一些变化。

  1. 动词的简体与敬体
    礼貌体 简体
    ます (原形)
    ました た形
    ません ない形
    ませんでした なかった
    • ある和行く变形特殊，需要注意。
    • でしたで 的简体是 だった
  2. 一类形容词的简体与敬体
    敬体：一类形容词（包括各种变形）+です
    简体：不加です
  3. 二类形容词的简体与敬体
    简体 敬体
    ～だ　　　 ～です
    ～じゃない ～ではありません
    ～だった ～でした
    ～じゃなかった ～ではありませんでした
    1. 名词简体（同二类形容词）
    2. 句子けど句子
      　表示转折或铺垫。
    • -昨日の試験、どうだった？
      -ちょっと難しかったけど、まあまあできたよ。
      昨天的考试怎么样？虽然有点难但还行吧。
    • 来週送别送別会をするけど、都合はどうかな？
      下周要办送别会，大家都有空来吧？
  4. (句子) って
    表示传闻，听说。
    • 太田、今度中国へ転勤だって。
      太田，听说这次你要调到中国了。
  5. ~方（かた）
    人的敬称。あの方：那位。
    动词词干（ます形去掉ます）+方表示方法。
    読みます → 読み方 (读法)
    使います → 使い方 (用法)
    話します → 話し方(说法)

第二十三课

  1. 动词たり 动词たり します
    表示两个动词并列。有时会做动作一，有时做动作二。用于例举。
    • 小野さんは休みの日、散歩したり　買い物に行ったりします。
      小野休息日有时候会去散步，有时候会去购物。
    • 部屋の中を行ったり来たりします。
      在房间里走来走去。（此处的を表示经过的场所）
  2. 一类形かったり 一类形かったり　です
    • 飛行機のチケットは　高かったり　安かったり　です。
      机票有的贵有的便宜。
      二类形だったり 二类形だったり　です
      この公園は時間によって　静かだったり　にぎやかだったり　です
      名词だったり 名词だったり　です
  3. 名词 か
好きです
